The Importance of CSS Grid

How does it help?

CSS Grid allows us to achieve a responsive design on all devices with as little as two lines of code. It increases the number of creative designs we can make and allows us to have unique websites.

What is it?

CSS Grid is a two-dimensional layout system created specifically to tackle grid-based user interfaces on the web. This can but used to make a full website layout or on a smaller scale as well. It also reduces the need for media queries for responsiveness, which can take up code.
